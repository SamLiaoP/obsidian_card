# ✅ 三大支柱（核心擇偶條件）

## 1. **🧠 能夠聊抽象問題**  
   他能和我一起思考關係的本質、情緒的結構、價值觀的落差與生命的意義。  
   他不怕複雜，甚至願意共同拆解。

## 2. **🪞 能夠反思自我**  
   他能察覺自己的情緒、防衛與慣性，願意把「我怎麼了」說出來，而不是只表現行為表層。  
   他知道誠實比聰明重要。

## 3. **🏗️ 有成長型思維**  
   他願意調整自己，不逃避困難、不固著當下。  
   我們可以一起變好、一起建構、一起更新彼此對關係的想像。

# ❤️ 關係需求與愛情觀

- 我渴望的不只是激情，而是**能共構、能對話、能一起變得更強**的關係。
- 我曾經在與Nissen的關係中感受到：「我可以極度誠實，完全無所顧忌地說出自己」，那成為我對關係的理想參照。
