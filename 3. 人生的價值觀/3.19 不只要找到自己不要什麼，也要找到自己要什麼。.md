
「如果不知道喜歡做什麼，先找不喜歡做的，找出共通性後，就有個至少知道哪裡不想去的指引」

但我後來覺得 即使從自己不喜歡什麼事情出發 也得找到自己喜歡什麼 而不要只停留在 我不喜歡這些東西 

因為反對會觸發負面情緒

「從反對開始沒有錯，因為拒絕能界定邊界；  
但若只會反對，你就永遠只能生活在邊界內。  
成長的關鍵，是從拒絕某物，走向擁抱某物。  
因為唯有支持，才能帶來穩定、積極與建設性的情緒能量。」

為了逃離某個地方，也只是回到「反對框架」內。
這會造成很大的心理負擔

| 面向   | 反對導向 (Oppose frame) | 支持導向 (Support frame) |
| ---- | ------------------- | -------------------- |
| 動機焦點 | 避免錯誤、逃離不適           | 追求滿足、實現價值            |
| 情緒傾向 | 容易焦慮、憤怒、厭倦          | 相對穩定、正向、持久           |
| 認知模式 | 聚焦缺陷、二元思維           | 聚焦可能性、整合思維           |
| 長期效果 | 容易耗竭與失落感            | 提升投入感與自我決定感          |
| 職涯結果 | 「我知道不想幹嘛」不斷逃跑，且迷茫   | 「我知道要去哪」而有方向         |
